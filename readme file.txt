TastyGo project is a .NET application developed in C#.

TastyGo.csproj (contains references to libraries and dependencies).
packages.config (lists NuGet packages used in the project).

-------------------------------------------------------------------------------------------------------------------------------

Libraries

Azure.Core (v1.38.0): Provides shared abstractions and helpers for Azure SDK libraries.

Azure.Identity (v1.11.4): Offers token authentication for Azure SDKs using Azure Active Directory.

Microsoft.Bcl.AsyncInterfaces (v1.1.1): Provides support for asynchronous programming patterns in older .NET frameworks.

Microsoft.Data.SqlClient (v5.1.6): A .NET library for interacting with SQL Server databases.

Microsoft.Data.SqlClient.SNI (v5.1.1): Enhances SQL Server network interface capabilities for better database communication.

Microsoft.Identity.Client: Supports Azure Active Directory authentication and provides token management for applications.
